# IPL Performance Analytics Dashboard

An interactive data analytics project built using Python and Plotly to analyze IPL batting performance.

## Key Insights
- Top run scorers in IPL
- Boundary comparison (Fours vs Sixes)
- Batting average analysis
- Century count by batsmen

## Tools Used
- Python
- Pandas
- Plotly

## Output
Interactive Plotly dashboard visualizing IPL batting trends.
